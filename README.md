# bunner-rs

Simple game utilizing [bevy](https://bevyengine.org/) game engine and [Entity Component System](https://en.wikipedia.org/wiki/Entity_component_system) architecture.

## How to run examples
```rust
cargo run --example 01_move_sprite 
```
